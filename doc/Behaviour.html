Class: Behaviour

Behaviour

This class represents a behaviour for a GameElement.

Constructor

new Behaviour(param)

Parameters:
Name Type Description
param Object The parameters of the behaviour. The object param contains these callbacks:
  • start : This function runs when the engine starts at first time.
  • update : This function runs each frame.
  • onEnterCollision : This function runs when the game element collides with other game element.
  • onExitCollision : This function runs when the collision is finished.
  • onDestroy : This function runs when the game element is destroyed.
